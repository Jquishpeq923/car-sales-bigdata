# 📊 Análisis y Predicción de Ventas Automotrices

Este repositorio contiene el código y la documentación para nuestro proyecto de análisis de datos y machine learning. 

## 🎯 Objetivos del Proyecto

* Explorar y estructurar el dataset histórico de ventas de vehículos.
* Identificar las marcas que registraron el mayor volumen de ventas.
* Analizar los datos para determinar qué marcas generaron las mayores comisiones.
* Desarrollar un modelo predictivo para estimar las ventas futuras de estas marcas mes a mes.
